This project is a modern, fully responsive **Glassmorphism** style Login Page, designed according to the latest 2026 web UI/UX trends.

## 🔗 Live Demo
> **[Click here for Live Demo](https://coderking-dev.github.io/Login-Ghost/)**

## 🚀 Key Features

*   ✨ **Glassmorphism Design:** High-quality frosted glass effect with backdrop-blur.
*   🖋️ **Floating Labels:** Modern input fields with dynamic label transitions.
*   👁️ **Password Toggle (JavaScript):** Show/Hide password functionality with dynamic icon switching.
*   📱 **Fully Responsive:** Perfectly optimized for Mobile, Tablet, and Desktop devices.
*   🎨 **Font Awesome 6.5.1:** Beautifully integrated vector icons for a premium look.
*   🌐 **Social Media Integration:** Elegant social login buttons for Google, Facebook, Instagram, and X (Twitter).

## 🛠️ Built With

*   **HTML5:** Semantic and clean markup.
*   **CSS3:** Advanced styling, including glassmorphism effects, animations, and media queries.
*   **JavaScript:** Password visibility logic and icon manipulation.
*   **Google Fonts:** Modern typography using the "Poppins" font family.

## 🖥️ Getting Started

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/CoderKing-Dev/Login-Ghost.git
   ```
2. Open the project folder and launch `index.html` in your favorite web browser.
